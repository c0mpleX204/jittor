# AutoDL 简化实验指令

所有命令默认在云平台执行：

```bash
cd /root/code2/release4/starter_code
source /root/jittor-venv/bin/activate
```

## 0. 数据盘

把实验输出切到数据盘：

```bash
bash tools/autodl_prepare_datadisk.sh /root/autodl-tmp/jittor_pcd
```

如果 `/root/code2/release4/data` 占系统盘，把它搬到数据盘：

```bash
mkdir -p /root/autodl-tmp/jittor_pcd
mv /root/code2/release4/data /root/autodl-tmp/jittor_pcd/data
ln -s /root/autodl-tmp/jittor_pcd/data /root/code2/release4/data
```

## 1. 生成本地测试集

生成固定 clean：

```bash
python tools/local_make_true_test.py \
  --dataset-dir dataset_train \
  --datalist datalist/train.txt \
  --out-dir true_test \
  --list-out datalist/true_test.txt \
  --num-models 200 \
  --points 50000 \
  --seed 2026 \
  --overwrite
```

生成固定 noisy：

```bash
python tools/local_make_noisy_test.py \
  --gt-dir true_test \
  --gt-list datalist/true_test.txt \
  --out-dir noisy_test \
  --list-out datalist/noisy_test.txt \
  --noise-type laplace \
  --std-min 0.005 \
  --std-max 0.020 \
  --seed 2026 \
  --overwrite
```

## 2. 训练

```bash
mkdir -p /root/autodl-tmp/jittor_pcd/result/logs

python -u run.py --task configs/task/train_vm_autodl_tmp.yaml \
  2>&1 | tee /root/autodl-tmp/jittor_pcd/result/logs/train_baseline.log
```

权重保存到：

```text
/root/autodl-tmp/jittor_pcd/experiments/vm/checkpoint_*.pkl
```

## 3. 推理多个 checkpoint

只做推理和整理，不打分。每个 epoch 的降噪点云会单独保存到 `result/epoch_xx/denoisy_test/`。

```bash
for E in 20 40 60 80; do
  echo "========== predict epoch ${E} =========="

  sed -i "s#^load_ckpt:.*#load_ckpt: /root/autodl-tmp/jittor_pcd/experiments/vm/checkpoint_${E}.pkl#" configs/task/predict_noisy_test_autodl_tmp.yaml
  sed -i "s#^  save_dir:.*#  save_dir: /root/autodl-tmp/jittor_pcd/result/epoch_${E}/denoisy_test_raw#" configs/task/predict_noisy_test_autodl_tmp.yaml

  rm -rf /root/autodl-tmp/jittor_pcd/result/epoch_${E}/denoisy_test_raw
  rm -rf /root/autodl-tmp/jittor_pcd/result/epoch_${E}/denoisy_test

  python -u run.py --task configs/task/predict_noisy_test_autodl_tmp.yaml \
    2>&1 | tee /root/autodl-tmp/jittor_pcd/result/logs/predict_epoch${E}.log

  python tools/local_collect_predictions.py \
    --raw-dir /root/autodl-tmp/jittor_pcd/result/epoch_${E}/denoisy_test_raw \
    --out-dir /root/autodl-tmp/jittor_pcd/result/epoch_${E}/denoisy_test \
    --list datalist/noisy_test.txt \
    --input-prefix noisy_test \
    --filename denoised.npy \
    --overwrite

  find /root/autodl-tmp/jittor_pcd/result/epoch_${E}/denoisy_test -name "denoised.npy" | wc -l
done
```

## 4. 打分和可视化多个 checkpoint

已经有 `result/epoch_xx/denoisy_test/` 后，单独打分和可视化：

```bash
for E in 20 40 60 80; do
  echo "========== score epoch ${E} =========="

  python tools/local_score_and_visualize.py \
    --gt-dir /root/autodl-tmp/jittor_pcd/true_test \
    --noisy-dir /root/autodl-tmp/jittor_pcd/noisy_test \
    --pred-dir /root/autodl-tmp/jittor_pcd/result/epoch_${E}/denoisy_test \
    --mesh-dir dataset_train \
    --result-dir /root/autodl-tmp/jittor_pcd/result/epoch_${E} \
    --workers 8 \
    --visualize-count 8 \
    --overwrite

  cat /root/autodl-tmp/jittor_pcd/result/epoch_${E}/scores/summary_cd_p2s_scores.json
done
```

如果只想打分、不生成可视化：

```bash
for E in 20 40 60 80; do
  python tools/local_score_and_visualize.py \
    --gt-dir /root/autodl-tmp/jittor_pcd/true_test \
    --noisy-dir /root/autodl-tmp/jittor_pcd/noisy_test \
    --pred-dir /root/autodl-tmp/jittor_pcd/result/epoch_${E}/denoisy_test \
    --mesh-dir dataset_train \
    --result-dir /root/autodl-tmp/jittor_pcd/result/epoch_${E} \
    --workers 8 \
    --visualize-count 0 \
    --overwrite
done
```

## 6. 官方测试集提交

选出最佳 checkpoint 后，例如 `checkpoint_60.pkl`：

```bash
sed -i "s#^load_ckpt:.*#load_ckpt: /root/autodl-tmp/jittor_pcd/experiments/vm/checkpoint_60.pkl#" configs/task/predict_submit_autodl_tmp.yaml
sed -i "s#^  save_dir:.*#  save_dir: /root/autodl-tmp/jittor_pcd/submit_result_raw#" configs/task/predict_submit_autodl_tmp.yaml

rm -rf /root/autodl-tmp/jittor_pcd/submit_result_raw
rm -rf /root/autodl-tmp/jittor_pcd/submit_result

python -u run.py --task configs/task/predict_submit_autodl_tmp.yaml \
  2>&1 | tee /root/autodl-tmp/jittor_pcd/result/logs/predict_submit_epoch60.log

python tools/local_collect_predictions.py \
  --raw-dir /root/autodl-tmp/jittor_pcd/submit_result_raw \
  --out-dir /root/autodl-tmp/jittor_pcd/submit_result \
  --list datalist/test.txt \
  --input-prefix dataset_test_noisy \
  --filename denoised.npy \
  --overwrite

cd /root/autodl-tmp/jittor_pcd/submit_result
zip -r /root/autodl-tmp/jittor_pcd/result.zip shapenet
```

最终提交：

```text
/root/autodl-tmp/jittor_pcd/result.zip
```
