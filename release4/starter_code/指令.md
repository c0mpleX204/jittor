# 本地固定测试集实验指令

下面命令默认你已经进入云算力平台的 `starter_code` 目录。

```bash
cd /你的路径/starter_code
```

如果云平台环境还没装依赖，先执行：

```bash
python -m pip install -r requirements.txt
```

## 1. 生成本地干净测试集 true_test

从 `dataset_train` 的 OBJ 网格里抽样，生成固定的 `true_test/shapenet/.../clean.npy`。

```bash
python tools/local_make_true_test.py \
  --dataset-dir dataset_train \
  --datalist datalist/train.txt \
  --out-dir true_test \
  --list-out datalist/true_test.txt \
  --num-models 200 \
  --points 50000 \
  --seed 2026 \
  --overwrite
```

说明：

- `--num-models 200` 可以改成 50、100、500。
- `--points 50000` 对齐赛题测试点云规模。
- `--overwrite` 会删除旧的 `true_test` 后重新生成。

## 2. 给 true_test 加噪，生成 noisy_test

```bash
python tools/local_make_noisy_test.py \
  --gt-dir true_test \
  --gt-list datalist/true_test.txt \
  --out-dir noisy_test \
  --list-out datalist/noisy_test.txt \
  --noise-type laplace \
  --std-min 0.005 \
  --std-max 0.020 \
  --seed 2026 \
  --overwrite
```

这一步会生成：

```text
noisy_test/shapenet/.../noisy.npy
noisy_test/noise_meta.csv
noisy_test/noise_meta.json
datalist/noisy_test.txt
```

## 3. 训练 baseline

```bash
mkdir -p result/logs

python -u run.py --task configs/task/train_vm.yaml 2>&1 | tee result/logs/train_baseline.log
```

训练完成后，权重默认在：

```text
experiments/vm/checkpoint_0.pkl
experiments/vm/checkpoint_1.pkl
...
experiments/vm/checkpoint_99.pkl
```

## 4. 对某个 checkpoint 推理 noisy_test

先改预测配置里的权重路径。比如测试第 99 个 epoch：

```bash
sed -i "s#^load_ckpt:.*#load_ckpt: experiments/vm/checkpoint_99.pkl#" configs/task/predict_noisy_test.yaml
```

删除旧预测并重新推理：

```bash
rm -rf denoisy_test_raw denoisy_test

python -u run.py --task configs/task/predict_noisy_test.yaml 2>&1 | tee result/logs/predict_epoch99.log
```

整理 baseline writer 的临时输出，生成干净的 `denoisy_test/shapenet/.../denoised.npy`：

```bash
python tools/local_collect_predictions.py \
  --raw-dir denoisy_test_raw \
  --out-dir denoisy_test \
  --list datalist/noisy_test.txt \
  --overwrite
```

## 5. 打分并生成可视化文件

```bash
python tools/local_score_and_visualize.py \
  --gt-dir true_test \
  --noisy-dir noisy_test \
  --pred-dir denoisy_test \
  --mesh-dir dataset_train \
  --result-dir result/epoch_99 \
  --workers 8 \
  --visualize-count 8 \
  --overwrite
```

输出位置：

```text
result/epoch_99/scores/summary_cd_p2s_scores.json
result/epoch_99/scores/per_sample_cd_p2s_scores.csv
result/epoch_99/visualization/
```

可视化文件是 `.ply`，可以用 CloudCompare 或 MeshLab 打开。

## 6. 一次测试多个 checkpoint

例如只测 20、40、60、80、99：

```bash
mkdir -p result/logs

for E in 20 40 60 80 99; do
  sed -i "s#^load_ckpt:.*#load_ckpt: experiments/vm/checkpoint_${E}.pkl#" configs/task/predict_noisy_test.yaml

  rm -rf denoisy_test_raw denoisy_test

  python -u run.py --task configs/task/predict_noisy_test.yaml 2>&1 | tee result/logs/predict_epoch${E}.log

  python tools/local_collect_predictions.py \
    --raw-dir denoisy_test_raw \
    --out-dir denoisy_test \
    --list datalist/noisy_test.txt \
    --overwrite

  python tools/local_score_and_visualize.py \
    --gt-dir true_test \
    --noisy-dir noisy_test \
    --pred-dir denoisy_test \
    --mesh-dir dataset_train \
    --result-dir result/epoch_${E} \
    --workers 8 \
    --visualize-count 8 \
    --overwrite
done
```

每次实验结束后，把整个 `result` 文件夹下载回 Windows 归档即可。
